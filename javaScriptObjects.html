JavaScript Objects

This demo is done with JavaScript Objects. I did not demonstrate classes, but since 2015 JavaScript has had classes available. The way classes inherit is different (using "Extends" to inherit, etc.) But I wanted to understand how to inherit from an OBJECT, not a class. So my code demonstration uses Objects only and no classes for inheritance, instantiation, properties, and methods.
